Usage Notes

Container

Use max-w-4xl mx-auto for main content containers.

Use p-6 for card padding, gap-4 for grid gaps.

Border Radius

Use rounded-xl for cards, rounded-lg for buttons and inputs.

Transitions

Always add transition-colors atau transition-all untuk interactive elements.

Icons
